## Appendix Package Purpose

## 附錄套件目的

### EN

The Final Appendix Package consolidates the operational reference materials of the Artibird SKU & Model Naming Specification v1.0.

This appendix package does not replace the main specification chapters. Instead, it provides fast-access tables, templates, abbreviations, examples, and checklists that can be copied into spreadsheets, ERP staging tables, PLM systems, certification trackers, Amazon preparation records, and release-control workflows.

最終附錄套件整合《Artibird SKU 與型號命名規範 v1.0》的運營參考材料。

它面向產品、工程、認證、生產、包裝、Amazon、渠道、支持與文件團隊的日常使用。

本附錄套件不取代正文規範章節，而是提供可快速查閱的表格、模板、縮寫、示例與檢查清單，可直接複製至 Excel、ERP 暫存表、PLM 系統、認證追蹤表、Amazon 準備記錄與發布管控流程中。

# Appendix A — Official SKU Structure

# 附錄 A｜官方 SKU 結構

## A.1 Standard SKU Format

## A.1 標準 SKU 格式

### EN

```text
ATB-[PRODUCT]-[GEN]-[REGION]-[CONNECTIVITY]-[POWER]-[VARIANT]-[PACKAGE]
```

### ZH-TW

```text
ATB-[產品]-[世代]-[區域]-[連接]-[供電]-[版本]-[包裝]
```

---

## A.2 SKU Field Definition

## A.2 SKU 欄位定義

|Field Order|EN Field|ZH-TW Field|Example|Meaning|
|--:|---|---|---|---|
|1|Brand Prefix|品牌前綴|ATB|Artibird|
|2|Product Line|產品線|ACT|Artibird Act|
|3|Generation|世代|G1|First generation|
|4|Region|區域|US|United States|
|5|Connectivity|連接方式|WF|Wi-Fi|
|6|Power|供電架構|AC|AC mains|
|7|Variant|版本 / 功能|E|Energy-aware|
|8|Package|包裝|S|Single unit|

---

## A.3 Standard Example

## A.3 標準示例

### EN

```text
ATB-ACT-G1-US-WF-AC-E-S
```

This means: Artibird Act, first generation, U.S. market, Wi-Fi, AC mains-powered, Energy-aware version, single-unit package.

### ZH-TW

```text
ATB-ACT-G1-US-WF-AC-E-S
```

表示：Artibird Act，第一代，美國市場，Wi-Fi，AC 市電供電，能源感知版本，單品包裝。

# Appendix C — Generation Code Table

# 附錄 C｜世代代碼表

|Generation Code|Meaning|Usage|
|---|---|---|
|G0|Pre-generation / experimental|Internal concept or prototype only|

|G3|Third generation|Next major generation|

**Rule｜規則：**  
Generation code must reflect product generation, not firmware version, PCB version, or package version.  
世代代碼反映產品世代，而不是韌體版本、PCB 版本或包裝版本。
