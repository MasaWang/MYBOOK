# 2. Catalog Statement

# 2. 型錄說明

## EN

This catalog presents the Artibird product layer as the physical expression of the OceanAI coordinated intelligent environment system.

Artibird products are not presented as isolated smart devices. They are presented as coordinated participants within a system architecture that connects runtime behavior, local control, device access, user interaction, sensing, execution, synchronization, cloud services, and Edge intelligence.

The purpose of this catalog is to help customers, channel partners, system integrators, sales teams, and deployment partners understand what each Artibird product does, how each product participates in OceanAI, and how different products can be combined into practical solutions for homes, classrooms, buildings, campuses, energy systems, vision-aware spaces, agriculture, and infrastructure environments.

## ZH-TW

本型錄展示 Artibird 產品層，作為 OceanAI 協同智慧空間系統在真實環境中的實體表現。

Artibird 產品不被呈現為孤立智慧設備，而是被呈現為一套系統架構中的協同參與者。這套架構連接 Runtime 行為、本地控制、設備接入、用戶互動、感知、執行、同步、雲端服務與 Edge 智慧。

本型錄的目的，是幫助客戶、渠道合作夥伴、系統整合商、銷售團隊與部署合作夥伴理解每一個 Artibird 產品做什麼、如何參與 OceanAI，以及不同產品如何組合為住宅、教室、建築、校園、能源系統、視覺感知空間、農業與基礎設施環境中的實際方案。

# 3. Executive Summary

# 3. 執行摘要

## EN

Artibird is the product and device layer of the OceanAI system.

Where OceanAI defines the system-level intelligence architecture, Artibird provides the physical products that allow this architecture to operate in real environments. Each Artibird product has a defined role within the coordinated intelligent environment: some products host local runtime services, some provide device access, some capture user intention, some sense environmental context, some execute physical actions, and others synchronize appliances or provide visual awareness.

This catalog introduces the Artibird product family as a complete product layer for coordinated spaces. It explains the relationship between Artibird products and the OceanAI architecture, summarizes each product line, and provides a foundation for sales communication, channel partnership, deployment planning, website content, product pages, and future PDF catalog design.

The central message is simple:

**Artibird makes OceanAI visible in the physical world.**

## ZH-TW

Artibird 是 OceanAI 系統中的產品與設備層。

OceanAI 定義系統級智慧架構，而 Artibird 提供使這套架構能夠在真實環境中運作的實體產品。每一個 Artibird 產品在協同智慧空間中都有明確角色：有些產品承載本地 Runtime 服務，有些提供設備接入，有些捕捉用戶意圖，有些感知環境上下文，有些執行物理動作，也有產品負責家電同步或視覺感知。

本型錄將 Artibird 產品家族作為面向協同空間的完整產品層進行介紹。它說明 Artibird 產品與 OceanAI 架構之間的關係，總結每一條產品線，並為銷售溝通、渠道合作、部署規劃、網站內容、產品頁與未來 PDF 型錄設計提供基礎。

# 5. Relationship to OceanAI

# 5. 與 OceanAI 的關係

## EN

OceanAI and Artibird are related but not identical.

OceanAI is the system-level intelligence architecture. It defines how devices, data, local runtime, cloud services, Edge intelligence, security, privacy, validation, and governance operate as one coordinated system.

Artibird is the product and physical device layer. It provides the actual devices, interfaces, gateways, sensing nodes, execution nodes, synchronization nodes, and user-facing control tools that allow the OceanAI architecture to operate in physical spaces.

In simple terms:

**OceanAI is the system. Artibird is the product layer that brings the system into real environments.**

## ZH-TW

OceanAI 與 Artibird 彼此相關，但並不相同。

OceanAI 是系統級智慧架構。它定義設備、數據、本地 Runtime、雲端服務、Edge 智慧、安全、隱私、驗證與治理如何作為一個協同系統共同運作。

Artibird 是產品與實體設備層。它提供實際設備、介面、網關、感知節點、執行節點、同步節點與面向用戶的控制工具，使 OceanAI 架構能夠在物理空間中運作。

簡言之：

**OceanAI 是系統。Artibird 是讓系統進入真實環境的產品層。**
