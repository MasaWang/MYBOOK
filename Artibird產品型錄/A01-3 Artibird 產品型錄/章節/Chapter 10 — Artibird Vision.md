# 第十章｜Artibird Vision

---

## 10.1 Product Positioning

## 10.1 產品定位

**EN**  
Artibird Vision is the **visual perception node** of the Artibird product system. It provides visual events, local recording, security awareness, AI-assisted visual context, and environmental observation for coordinated intelligent environments.

Within the OceanAI architecture, Artibird Vision belongs primarily to the **Sensing / Edge Context Layer**. It allows spaces to interpret visual signals and integrate them into runtime behavior, automation, scene execution, and Edge-assisted AI optimization.

**ZH-TW**  
Artibird Vision 是 Artibird 產品系統中的**視覺感知節點**。它提供視覺事件、本地錄影、安全感知、AI 輔助視覺上下文，以及環境觀測功能，用於協同智慧空間。

在 OceanAI 架構中，Artibird Vision 主要屬於**感知 / Edge 上下文層**。它使空間能解讀視覺信號，並將其整合進 Runtime 行為、自動化、場景執行及 Edge 輔助 AI 優化。

---

## 10.2 System Role

## 10.2 系統角色

**EN**  
Artibird Vision acts as a visual input device for the OceanAI runtime system. It provides security awareness, occupancy detection, motion events, room monitoring, and visual context for automation or Edge-assisted AI interpretation.

Vision devices may interface with Hub / ArtiOS, App, Cloud, Edge, Panel, Switch, Act, or other supported devices. Its role is to provide actionable visual data without replacing the device, ensuring safety, privacy, and coordination under supported deployment conditions.

**ZH-TW**  
Artibird Vision 作為 OceanAI Runtime 系統的視覺輸入設備。它提供安全感知、佔用偵測、移動事件、房間監測，以及用於自動化或 Edge 輔助 AI 解讀的視覺上下文。

Vision 設備可與 Hub / ArtiOS、App、Cloud、Edge、Panel、Switch、Act 或其他受支援設備介面。它的角色是在支援部署條件下，提供可行的視覺數據，並確保安全、隱私與協同，不取代設備本身。

## 10.3 Core Functions

## 10.3 核心功能

**EN**  
Artibird Vision provides visual context and security awareness. Its functions may include motion detection, occupancy detection, AI-assisted visual events, local recording, alerts, and interaction with automation, scenes, or runtime logic.

Visual perception should always remain bounded by camera model, firmware version, installation position, privacy configuration, network path, and validation status.

**ZH-TW**  
Artibird Vision 提供視覺上下文與安全感知。其功能可包括移動偵測、佔用偵測、AI 輔助視覺事件、本地錄影、警報，以及與自動化、場景或 Runtime 邏輯互動。

視覺感知必須始終受到攝影機型號、韌體版本、安裝位置、隱私配置、網路路徑與驗證狀態約束。

## 10.4 Key Product Value

## 10.4 核心產品價值

**EN**  
Artibird Vision gives coordinated intelligent environments the ability to perceive visual events. It transforms rooms, classrooms, offices, and shared spaces into spaces capable of security monitoring, AI-assisted interpretation, scene automation, and informed system decision-making.

Its value increases when integrated with Hub, Edge, Cloud, Panel, Switch, Act, Sense, and Sync to provide comprehensive space awareness and automation input.

**ZH-TW**  
Artibird Vision 為協同智慧空間提供感知視覺事件的能力。它將房間、教室、辦公室與共享空間轉化為能夠進行安全監控、AI 輔助解讀、場景自動化與系統決策的智慧空間。

當與 Hub、Edge、Cloud、Panel、Switch、Act、Sense 與 Sync 整合時，其價值會增加，提供完整空間感知與自動化輸入。
