# Chapter 1 — Artibird Product Layer Overview

# 第一章｜Artibird 產品層總覽

---

## 1.1 What Is Artibird

## 1.1 Artibird 是什麼

**EN**  
Artibird is the **product and physical device layer** of the OceanAI coordinated intelligent environment system. It provides the tangible, deployable devices that allow OceanAI’s system-level intelligence to operate in real environments.

Artibird is not a collection of isolated smart devices. Each product is designed to participate in a **coordinated runtime environment**, contributing to sensing, execution, interaction, synchronization, or local runtime hosting.

**ZH-TW**  
Artibird 是 OceanAI 協同智慧空間系統中的**產品與實體設備層**。它提供可部署的實體設備，使 OceanAI 的系統級智慧能夠在真實環境中運作。

Artibird 不是孤立的智慧設備集合。每一個產品都被設計為參與**協同 Runtime 環境**，在感知、執行、互動、同步或本地 Runtime 承載中扮演角色。

---

## 1.2 Artibird and OceanAI

## 1.2 Artibird 與 OceanAI

**EN**  
OceanAI defines the **system-level intelligence architecture**, governing runtime behavior, local-cloud-edge coordination, device integration, security, privacy, validation, and governance.

Artibird provides the **physical manifestation** of this architecture: hubs, gateways, switches, panels, execution nodes, sensing nodes, synchronization devices, visual perception products, and the user-facing App. Together, they allow spaces to sense, respond, execute, and synchronize as a coordinated intelligent environment.

**ZH-TW**  
OceanAI 定義了**系統級智慧架構**，管理 Runtime 行為、本地—雲端—Edge 協同、設備整合、安全、隱私、驗證與治理。

Artibird 提供這套架構的**實體呈現**：Hub、Gate、Switch、Panel、Act、Sense、Sync、Vision 與用戶 App。這些產品共同使空間能夠感知、回應、執行與同步，形成協同智慧空間。

---

## 1.3 From Devices to Coordinated Environments

## 1.3 從設備到協同空間

**EN**  
Artibird products should not be described as isolated devices. They are **runtime participants**, each fulfilling a specific role that contributes to a **coordinated intelligent environment**.

The value of Artibird emerges from the combination of multiple devices participating in shared runtime behavior, enabling features like automation, scene execution, appliance synchronization, energy management, environmental sensing, visual awareness, and local/cloud intelligence.

**ZH-TW**  
Artibird 產品不應被描述為孤立設備。它們是**Runtime 參與者**，每個產品都有特定角色，共同構成**協同智慧空間**。

Artibird 的價值源自多個設備在共享 Runtime 行為中的協同作用，實現自動化、場景執行、家電同步、能源管理、環境感知、視覺感知，以及本地與雲端智慧能力。

---

## 1.4 Product Family Summary

## 1.4 產品家族總覽

**EN**  
The Artibird product family is organized to cover all functions required by a coordinated intelligent environment. Each product is designed to perform specific roles while contributing to the larger system.

|Product|Core Role|System Participation|
|---|---|---|
|Artibird Hub|Local runtime center|Hosts local services, state, automation, and ArtiOS modules|
|Artibird Gate|Device access gateway|Connects supported devices to the local runtime environment|
|Artibird Sync|Appliance synchronization node|Brings IR / RF appliances into coordinated scenes and automations|
|Artibird Switch|Smart switch and event node|Captures user intention and supports scene or relay behavior|
|Artibird Panel|Room-level interaction interface|Provides shared room control, scene selection, and status feedback|
|Artibird Act|Electrical execution node|Performs relay, plug, load control, and optional energy measurement|
|Artibird Sense|Environmental sensing node|Provides temperature, humidity, light, occupancy, and context input|
|Artibird Vision|Visual perception node|Provides visual events, security awareness, and AI-assisted visual context|
|Artibird App|User control interface|Supports setup, local control, remote access, configuration, and status viewing|

**ZH-TW**

|產品|核心角色|系統參與|
|---|---|---|
|Artibird Hub|本地 Runtime 中樞|承載本地服務、狀態、自動化與 ArtiOS 模組|
|Artibird Gate|設備接入網關|將受支援設備接入本地 Runtime 環境|
|Artibird Sync|家電同步節點|將 IR / RF 家電納入協同場景與自動化|
|Artibird Switch|智慧開關與事件節點|捕捉用戶意圖，支援場景或繼電器行為|
|Artibird Panel|房間級互動介面|提供共享房間控制、場景選擇與狀態回饋|
|Artibird Act|電力執行節點|執行繼電器、插座、負載控制與可選電能量測|
|Artibird Sense|環境感知節點|提供溫度、濕度、光照、人在與上下文輸入|
|Artibird Vision|視覺感知節點|提供視覺事件、安全感知與 AI 輔助視覺上下文|
|Artibird App|用戶控制介面|支援設定、本地控制、遠端存取、配置與狀態查看|
