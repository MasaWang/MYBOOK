# 3. Product Commercial Overview

# 3. 產品商業總覽

## EN

Artibird Panel is a room-level control and interaction product within the Artibird ecosystem. It provides a dedicated interface for users to access scenes, devices, room modes, environmental status, and system actions in coordinated intelligent environments.

In homes, classrooms, offices, hotels, campuses, and commercial buildings, Artibird Panel creates a stable control point for spaces where users need more than a single switch. It can present scene shortcuts, room controls, lighting modes, climate-related controls, device groups, and system status depending on product variant and deployment configuration.

When paired with Artibird Hub and ArtiOS, Panel can participate in local runtime behavior. A touch action on the Panel may trigger a scene, control supported lighting, activate climate logic, interact with Artibird Act, receive context from Artibird Sense, coordinate with Artibird Sync, or display status from compatible devices. When connected to OceanAI Cloud, the system may support remote configuration, account-level synchronization, lifecycle management, and multi-site consistency.

Artibird Panel follows a local-first, cloud-enhanced model. Supported room controls and scene actions are designed to operate locally when required system components are present. Cloud services can enhance configuration, synchronization, remote management, and large-scale deployment visibility.

Artibird Panel should not be presented as a generic tablet, display, or standalone control screen. It is a dedicated room-level interaction node for Artibird coordinated environments.

## ZH-TW

Artibird Panel 是 Artibird 生態中的房間級控制與互動產品。它為用戶提供專用介面，用於在協同智慧空間中存取場景、設備、房間模式、環境狀態與系統動作。

在住宅、教室、辦公室、酒店、校園與商業建築中，Artibird Panel 為需要超越單一開關控制的空間建立穩定控制點。依產品版本與部署配置不同，它可呈現場景快捷、房間控制、照明模式、空調相關控制、設備群組與系統狀態。

當與 Artibird Hub 和 ArtiOS 配合使用時，Panel 可參與本地 Runtime 行為。Panel 上的一次觸控可觸發場景、控制受支援照明、啟動空調邏輯、與 Artibird Act 互動、接收 Artibird Sense 上下文、協同 Artibird Sync，或顯示相容設備狀態。當接入 OceanAI Cloud 後，系統可支援遠端配置、帳號級同步、生命週期管理與多站點一致性。

Artibird Panel 採用本地優先、雲端增強模式。在所需系統元件具備的條件下，受支援房間控制與場景動作設計為可本地運行。雲端服務可進一步增強配置、同步、遠端管理與大規模部署可視化。

Artibird Panel 不應被表述為普通平板、顯示器或獨立控制螢幕。它是面向 Artibird 協同環境的專用房間級互動節點。

## 4.3 Long Description

## 4.3 長描述

**EN**  
Artibird Panel is designed for spaces that need a clear, visible, and shared control surface. Instead of relying only on mobile apps or individual switches, Panel provides a dedicated room interface where users can trigger scenes, manage devices, view status, and interact with the Artibird runtime system.

In a complete Artibird deployment, Panel works with Hub, ArtiOS, Gate, Switch, Act, Sense, Sync, Vision, OceanAI Cloud, and OceanAI Edge. A user may select a classroom mode, activate a meeting mode, adjust a lighting scene, trigger appliance synchronization, or access environmental information depending on system configuration.

Artibird Panel is especially suitable for rooms where multiple users need access to the same controls, such as classrooms, meeting rooms, hotel rooms, offices, shared residential spaces, and commercial areas. It improves usability by making system control visible and accessible at the room level.

Artibird Panel does not make universal compatibility or unlimited control claims. Supported scenes, devices, interfaces, display functions, touch reliability, network behavior, and remote-management features must be defined by product variant and validated under specified conditions.

# 12. Commercial Claim Policy

# 12. 商業宣稱政策

## EN

All Artibird Panel commercial claims must remain consistent with the related engineering specification and validation results. Commercial materials must not use absolute claims such as “controls everything,” “works with all devices,” “unlimited scenes,” “universal wall panel,” “zero configuration,” or “guaranteed compatibility.”

Panel-related claims must distinguish between scene panels, display panels, wall-mounted panels, low-voltage panels, powered panels, and region-specific installation variants. Supported scenes, devices, UI functions, display content, touch reliability, wireless behavior, power method, mounting requirements, and cloud configuration features must be defined by product variant.

## ZH-TW

所有 Artibird Panel 商業宣稱必須與對應工程規格書與驗證結果保持一致。商業資料不得使用「控制所有設備」「支援所有設備」「無限制場景」「通用牆面面板」「零配置」「保證相容」等絕對化宣稱。

Panel 相關宣稱必須區分場景面板、顯示面板、牆面式面板、低壓面板、供電型面板與不同地區安裝版本。受支援場景、設備、UI 功能、顯示內容、觸控可靠性、無線行為、供電方式、安裝要求與雲端配置功能，必須依產品版本定義。
