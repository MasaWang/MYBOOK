# 3. Product Commercial Overview

# 3. 產品商業總覽

## EN

Artibird Sync is an IR / RF synchronization node designed for the Artibird product ecosystem. It allows supported appliances such as air conditioners, TVs, fans, set-top boxes, projectors, audio systems, curtains, and selected RF-controlled devices to participate in coordinated intelligent environments.

Built on an ESP32-based IR + RF architecture, Artibird Sync supports appliance control through infrared transmission, IR learning where supported, code-profile matching, and selected RF signal transmission depending on product variant. Its commercial value lies in extending the Artibird system to appliances that were not originally designed as smart devices.

In residential, classroom, office, hotel, campus, and commercial building environments, Artibird Sync can help connect traditional appliances to scenes, schedules, environmental triggers, energy logic, and remote-control workflows. When paired with Artibird Hub, ArtiOS, Sense, Act, and OceanAI Cloud, Sync becomes part of a broader coordinated system where appliance behavior can respond to time, sensor data, user interaction, and system state.

Artibird Sync follows a local-first, cloud-enhanced model. Supported local commands and scene actions can operate within the local environment when required local components are available. Cloud services may enhance the system through remote control, code library updates, lifecycle management, synchronization, usage visibility, and multi-site support.

Artibird Sync should not be presented as a universal controller for all appliances. Compatibility depends on validated IR protocols, RF frequency and coding support, appliance profiles, learning capability, placement, signal coverage, and product variant.

## ZH-TW

Artibird Sync 是一款面向 Artibird 產品生態設計的 IR / RF 同步節點。它使受支援家電，例如空調、電視、風扇、機上盒、投影機、音響系統、窗簾以及部分 RF 控制設備，能夠參與協同智慧空間。

Artibird Sync 基於 ESP32 的 IR + RF 架構，依產品版本支援紅外線發射、受支援條件下的 IR 學習、碼值配置匹配，以及部分 RF 信號發射。它的商業價值在於將原本並非智慧設備的家電延伸納入 Artibird 系統。

在住宅、教室、辦公室、酒店、校園與商業建築環境中，Artibird Sync 可協助傳統家電接入場景、排程、環境觸發、能源邏輯與遠端控制流程。當它與 Artibird Hub、ArtiOS、Sense、Act 以及 OceanAI Cloud 配合使用時，Sync 成為更大協同系統的一部分，使家電行為可根據時間、感測數據、用戶互動與系統狀態作出回應。

Artibird Sync 採用本地優先、雲端增強模式。在所需本地元件具備的條件下，受支援本地指令與場景動作可在本地環境中運作。雲端服務則可透過遠端控制、碼庫更新、生命週期管理、同步、使用可視化與多站點支援增強系統能力。

Artibird Sync 不應被表述為支援所有家電的通用控制器。其相容性取決於已驗證 IR 協議、RF 頻率與編碼支援、家電碼值配置、學習能力、放置位置、信號覆蓋與產品版本。

# 12. Commercial Claim Policy

# 12. 商業宣稱政策

## EN

All Artibird Sync commercial claims must remain consistent with the related engineering specification and validation results. Commercial materials must not use absolute claims such as “controls all appliances,” “universal remote for everything,” “works with every air conditioner,” “guaranteed learning,” or “always successful RF control.”

Compatibility-related claims must reference validated device categories, supported IR protocols, verified RF frequency and coding types, code-profile coverage, learning performance, and signal placement requirements. Terms such as “broad appliance support,” “IR learning,” “RF control,” or “AC control” should only be used with clear compatibility boundaries.

## ZH-TW

所有 Artibird Sync 商業宣稱必須與對應工程規格書與驗證結果保持一致。商業資料不得使用「控制所有家電」「支援所有設備的萬能遙控器」「支援每一台空調」「保證學習成功」「RF 控制永遠成功」等絕對化宣稱。

相容性相關宣稱必須引用已驗證設備類別、受支援 IR 協議、已驗證 RF 頻率與編碼類型、碼值配置覆蓋、學習表現與信號放置要求。「廣泛家電支援」「IR 學習」「RF 控制」「空調控制」等表述，只有在具備清晰相容性邊界時才可使用。
